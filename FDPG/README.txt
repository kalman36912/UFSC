The files of this folder can generate the results of the following paper:

    Seyed Saman Saboksayr and Gonzalo Mateos,
    "Accelerated graph learning from smooth signals,"
    IEEE Signal Processing Letters (2021).
    Preprint: https://arxiv.org/abs/2110.09677



    Files:
            sim_ER_200node.m    : Simulation for ER graph with 200 node (Fig. 1 (a))
            sim_ER_400node.m    : Simulation for ER graph with 400 node (Fig. 1 (a))
            sim_SBM_200node.m   : Simulation for SBM graph with 200 node (Fig. 1 (a))
            sim_SBM_400node.m   : Simulation for SBM graph with 400 node (Fig. 1 (a))
            sim_brain_66nodes.m : Simulation for brain data (Fig. 1 (b))
            sim_brain_time.m    : Simulation for wall-clock time in the result section
            sim_minnesota.m     : Simulation for Minnesota road network (Fig. 1 (c))
            sim_obj.m           : Simulation for dual suboptimality (Fig. 2)

    Note: The citation for the datasets that have been used in this study are as follows,

            Brain data:

            P. Hagmann, L. Cammoun, X. Gigandet, R. Meuli, C. J. Honey, V. J.
            Wedeen, and O. Sporns, “Mapping the structural core of human cerebral cortex,”
            PLoS Biol, vol. 6, no. 7, p. e159, 2008.

            Minnesota road network:

            T. A. Davis and Y. Hu, “The University of Florida Sparse Matrix Collection,”
            ACM Trans. Math. Softw., vol. 38, no. 1, Dec. 2011
